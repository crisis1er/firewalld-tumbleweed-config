# Support

## Before asking

- Read the full [README.md](README.md)
- Check [CHANGELOG.md](CHANGELOG.md) for recent changes
- Search existing [Issues](https://github.com/crisis1er/firewalld-tumbleweed-config/issues)
  and [Discussions](https://github.com/crisis1er/firewalld-tumbleweed-config/discussions)

## Where to get help

| Channel | Use for |
|---------|---------|
| [GitHub Issues](https://github.com/crisis1er/firewalld-tumbleweed-config/issues) | Bug reports, errors, unexpected behavior |
| [GitHub Discussions](https://github.com/crisis1er/firewalld-tumbleweed-config/discussions) | General questions, configuration help, ideas |
| [safeitexperts.com](https://safeitexperts.com) | In-depth articles and guides |
| [firewalld documentation](https://firewalld.org/documentation/) | Official firewalld documentation |
| [openSUSE Wiki — firewalld](https://en.opensuse.org/firewalld) | Official openSUSE firewalld documentation |

## What to include when asking

- Your openSUSE version: `cat /etc/os-release`
- firewalld version: `firewall-cmd --version`
- Exact error message — paste text, never screenshot
- What you already tried

## Compatibility
