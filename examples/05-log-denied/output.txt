# Reading the log fields:
# IN=    — interface the packet arrived on
# SRC=   — source IP (attacker / scanner)
# DST=   — destination IP (your machine)
# DPT=   — destination port being probed
# PROTO= — protocol (TCP/UDP/ICMP)
# SYN    — TCP connection attempt (new connection)
#
# Common DPT values seen in the wild:
#   22    — SSH brute-force attempts
#   3389  — RDP (Windows Remote Desktop) scanners
#   23    — Telnet scanners
#   80    — HTTP probes
#   8080  — Alternate HTTP scanners
#
# These are dropped silently by the public zone (target: DROP)
# LogDenied=all in firewalld.conf ensures every dropped packet is logged
